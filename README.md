# GILA CHALLENGE

by Fernando Paredes


## Tech Stack
* React 

### Prerequisities
* [npm](https://www.npmjs.com/)
* [node](https://nodejs.org/en/)

## Getting Started

### Install
Enter to the root folder and run the following command
```shell
npm install
```

### Running the challenge
```shell
npm start
```

### Run tests
```shell
npm test
```

The app runs at the url http://localhost:3000
